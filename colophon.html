A Static Angular 2 Site for GitHub Pages

Websites hosted on GitHub Pages are, by nature, static sites. The common convention for most sites on GitHub is to be based on Jekyll. Individual pages are.

  • Use CDN for script tags in index page
  • node_modules not stored in repository (so can't rely on them on host)
  • typescript files are stored in the repo, but I only want compiled javascript - set TypeScript compiler to pre-compile
  • GitHub hides anything in a folder starting with an underscore
